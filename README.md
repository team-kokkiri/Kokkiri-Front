# kokkiri-front

Kokkiri-front는 Vue 3 기반으로 개발되는 웹 프론트엔드 애플리케이션입니다. 실시간 통신과 인증 기능을 제공하는 인터페이스 구현을 목표로 하며, UI 구성에는 **Vuetify**, 페이지 이동에는 **Vue Router**를 사용할 예정입니다.

## 요구 사항
* **Node.js 16** 이상 권장
* 의존성 설치: `npm install`

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
